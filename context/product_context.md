# Product Context

## What We're Selling

**AgentForge** — Enterprise AI Agent Development, Without the Engineers

We build custom AI agents for enterprises. Not chatbots. Not copilots. **Autonomous agents** that do real work: process invoices, qualify leads, handle support tickets, manage inventory, negotiate contracts.

The twist? **We don't use human developers.**

Our agents are built by other agents. A customer describes what they need in plain English, and our Agent Builder Agent designs, codes, tests, and deploys a production-ready AI agent — typically in hours, not months.

This is recursive automation: **agents building agents building agents.**

Every enterprise wants AI agents. The bottleneck isn't desire — it's delivery.

**Our approach:**
1. Customer describes what they want
2. Agent Builder Agent creates the agent

No consultants. No sprints. No "we'll circle back on that."
